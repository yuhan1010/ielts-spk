Pause (Fluency)

- fillers

  • er / em ( 少說呀，別天天掛嘴上 )
  • well / you know / I mean / maybe / perhaps / I guess (I’m not sure) / like / kind of / sort of
  • Actually / in fact / to be honest
  • let me see / what I want to say is
  • I don't know (maybe)

- Language hesitation

  • What's the word? / what's it called?
  • Sorry I forgot the word.
  • Anyway, it's like + description.
  • It doesn't matter, anyway...

- content hesitation

  • It's a tough one, let me think about it.
  • To be honest, I've never thought about this before.
